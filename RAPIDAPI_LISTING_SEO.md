# RapidAPI Listing SEO — copy/paste

Optimized name, description, and tags for the PDF Forge API marketplace listing.
Target keywords developers actually search: **html to pdf api, pdf generation api,
invoice api, merge pdf api, convert html to pdf, pdf generator**.

---

## 1) API Name

RapidAPI's Name is the single most weighted field for search (it becomes the page title /
H1). Putting your top keyword in it — while keeping the "PDF Forge" brand — is worth it:

**Recommended:**
```
PDF Forge - HTML to PDF, Invoice & Merge API
```

Alternatives:
```
PDF Forge - HTML to PDF & Invoice Generator API
PDF Forge API - HTML to PDF, Invoices, Merge
```

⚠️ Before saving: changing the display Name usually does NOT change your API host/slug
(`pdf-forge-api.p.rapidapi.com`), but confirm the host is unchanged after you save — it's
baked into your backend base URL and docs. If it would change the slug, keep the current
name and rely on the description + tags below.

---

## 2) Short description (marketplace search-card one-liner)

This shows on search/browse cards — lead with keywords + a benefit. Pick one:

**A (recommended):**
```
Convert HTML to PDF, generate invoices from JSON, and merge PDFs — a fast, developer-friendly REST API. Pixel-perfect output, free tier, no setup.
```

**B (shorter):**
```
HTML to PDF, JSON to invoice, and merge PDFs via one simple REST API. Fast, pixel-perfect, free tier.
```

**C (benefit-led):**
```
Generate PDFs in one API call — convert HTML to PDF, build invoices from JSON, merge documents. Free tier, no rendering infra to run.
```

## 3) Overview opening paragraph (top of the Docs/long description)

Front-load keywords in the first 1–2 sentences (what Google shows in snippets):

```
PDF Forge is a fast HTML-to-PDF and PDF generation API. Convert any HTML into a
pixel-perfect PDF, generate styled invoices and receipts from JSON, and merge multiple
PDFs into one — all through a simple REST API, with a free tier and no rendering

and "export to PDF" features in your own app.
```

## Where each piece goes
- **Name** → General tab, API Name field.
- **Short description** → General tab, description/tagline field (the search-card text).
- **Overview paragraph** → Docs tab, top of the description (replaces the first paragraph
  of the Overview you pasted from RAPIDAPI_DOCS.md).
- **Tags** → General tab, Tags/keywords field.
- **Category** → General tab, Category.
