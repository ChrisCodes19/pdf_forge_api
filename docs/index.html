Frequently asked questions

How do I convert HTML to PDF with an API?

Send a POST request to /v1/html-to-pdf with your HTML in a JSON body. PDF Forge renders it with headless Chromium and returns a PDF file in the response — your CSS, fonts, and backgrounds come out exactly as they look in a browser.

Is there a free HTML to PDF API?

Yes. The free Basic plan includes 50 PDFs per month across all endpoints, so you can convert HTML to PDF, generate invoices, and merge PDFs at no cost while you evaluate.

Can I generate invoice PDFs from JSON?

Yes. POST your company, customer, and line-item data as JSON to /v1/invoice and PDF Forge calculates the subtotal, tax, and total, then returns a styled invoice PDF.

How do I merge PDFs via API?

POST an array of base64-encoded PDFs to /v1/merge and PDF Forge combines them into a single PDF, preserving order.

Which programming languages are supported?

Any language that can make an HTTP request — JavaScript/Node.js, Python, PHP, Ruby, Go, Java, C#, and more. It is a standard REST API.
